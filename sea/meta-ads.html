Meta's algorithm is genuinely good at finding people likely to convert — which means the biggest performance variable is almost always creative, not targeting. Accounts that plateau usually have the same three ad creatives running for six months, not a targeting problem.

We treat creative as something to test continuously, not something to set once. New hooks, new formats, new angles, tested in small batches against the existing best performer. Most weeks we're running at least two or three active creative tests across an account.

We're also realistic about iOS tracking limitations. Server-side tracking via the Conversions API recovers a meaningful chunk of the data Apple's privacy changes took away — we set this up for every Meta account we manage, not as an optional extra.

Meta recommends 50 conversion events per ad set per week. For most businesses that means at least €30–50 per day per campaign. Below that, results are unreliable.
